# Luminary-Auras-Reffect
A more visible UI element for Guardian's Luminary Elite Specialization

- Tracks Luminary's blessing.
- Tracks currently equipped Radiant Weapon buff, in and out of Forge.

<img width="438" height="252" alt="image" src="https://github.com/user-attachments/assets/1077c2de-01e7-4034-90a6-a5598e1e1c82" />

- If Sovereign of Light is equipped, additionally tracks Light Aura, so you know when to detonate it.

<img width="406" height="179" alt="image" src="https://github.com/user-attachments/assets/bb7a9517-3eb2-4613-93e5-16e16887ac31" />

- Tracks Empowered Armaments.

<img width="395" height="225" alt="image" src="https://github.com/user-attachments/assets/3e106a78-25eb-48d0-92c3-26678556a484" />

Additionally, turns the separator between F3 and F4 black to be consistent with other elite specs.


# How to use
First, download the zip file to your reffect root folder like so:

<img width="513" height="199" alt="image" src="https://github.com/user-attachments/assets/93e62597-d289-4add-98fb-da4da4f4612c" />

Then right click > Extract here (the necessary files will be added to the packs and icons subfolders).

Then, return to reffect ingame and refresh packs.

Finally, enable / disable whichever interface size you prefer:
